# Contributing

Thanks for contributing to SkillGraph Tutor.

## Setup

```bash
make setup
```

## Workflow

1. Create a branch.
2. Add tests for behavior changes.
3. Run `make fmt lint test demo`.
4. Open a PR with rationale and sample output.

## Standards

- Python 3.11+
- Keep offline mode first-class
- No API keys or secrets in repository
